# Fingerprint Collector

Simple HTML page to collect browser fingerprint data.

## Usage

Open: https://evheniu1993.github.io/fingerprint-collector/

## Features

- Canvas fingerprint
- WebGL fingerprint  
- Fonts detection
- Screen resolution
- Media devices
- Speech synthesis
- Sensors
- Battery API
- WebRTC leak test
- And more...

## Export

Click "Download JSON" to get full fingerprint data.
